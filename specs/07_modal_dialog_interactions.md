# Test: Modal Dialog Interactions

## Description
Test opening, interacting with, and closing modal dialogs.

## Steps

1. Navigate to https://the-internet.herokuapp.com/entry_ad
2. Wait for the modal dialog to appear (may take a few seconds)
3. Verify the modal is visible and contains a title
4. Verify the modal has a close button
5. Click the close button on the modal
6. Verify the modal disappears from the page
7. Verify the page content underneath is now accessible
8. Take a screenshot of the page after modal closes

## Expected Outcome
- Modal appears automatically after page load
- Modal content is accessible
- Modal can be closed successfully
- Page content is accessible after modal closes
- Timing and waiting work correctly
