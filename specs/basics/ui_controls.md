# Test: UI Controls (Checkboxes & Dropdowns)

## Description
Test interaction with standard UI controls like checkboxes and dropdown menus.

## Steps

1. Navigate to https://the-internet.herokuapp.com/checkboxes
2. Verify the heading "Checkboxes" is visible
3. Check "checkbox 1" (if not already checked)
4. Uncheck "checkbox 2"
5. Verify "checkbox 1" is checked
6. Verify "checkbox 2" is unchecked
7. Navigate to https://the-internet.herokuapp.com/dropdown
8. Verify the heading "Dropdown List" is visible
9. Select "Option 1" from the dropdown
10. Verify "Option 1" is currently selected
11. Select "Option 2" from the dropdown
12. Verify "Option 2" is currently selected

## Expected Outcome
- Can successfully toggle checkboxes
- Can successfully select dropdown options
- State verification works for these controls
